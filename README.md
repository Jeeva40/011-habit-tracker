# HabitSystems 🏆

**HabitSystems** is a modern, feature-rich Habit Tracker web application designed to help users build positive habits, stay consistent, and achieve their personal goals through analytics, gamification, and an intuitive user experience.

## ✨ Features

* ✅ Create, edit, delete, archive, and favorite habits
* 📅 Daily, weekly, monthly, and yearly habit tracking
* 🔥 Streak tracking with longest streak records
* 📊 Interactive dashboard with analytics
* 📈 GitHub-style habit heatmap
* 🎯 Goals and progress tracking
* 🏅 Achievements, XP, levels, and rewards
* 📖 Habit journal
* 💧 Water intake tracker
* 😊 Daily mood tracker
* ⏱️ Built-in Pomodoro timer
* 🔔 Habit reminders and notifications
* 🌙 Light & Dark Mode
* 🔍 Search, filter, and sort habits
* 💾 Local Storage support
* 📤 Import & Export (JSON/CSV)
* 📱 Fully responsive design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Local Storage API
* Font Awesome
* Google Fonts
